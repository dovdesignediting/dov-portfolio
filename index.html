2. שירותי תחזוקה חודשית (ריטיינר): במידה והלקוח בחר בחבילת תחזוקה (/$250חודש), השירות יכלול: • גיבוי מלא לאתר אחת לשבוע. • עדכוני ליבה, תוספים ותבניות (באתרי WordPress). • ניטור אבטחה בסיסי למניעת פריצות. • עד שעתיים חודשיות לשינויי תוכן קלים (הזנת טקסט/החלפת תמונה) שאינם דורשים עיצוב או פיתוח חדש. הערה: שעות התחזוקה אינן נצברות מחודש לחודש.

3. החרגות (מה לא כלול): • עלויות צד ג': תשלומים עבור דומיין, אחסון )Hosting), רישיונות לתוספים(Plugins)או שירותי API חיצוניים חלים על הלקוח .בלבד • תוכן: העלאת מוצרים/פוסטים מעבר למוגדר בהצעה תתומחר בנפרד. • שיחזור נתונים: שיחזור אתר שנפרץ או קרס עקב רשלנות לקוח (שלא תחת חבילת תחזוקה) יחויב בתשלום נפרד לפי שעות עבודה.

4. קניין רוחני ומסירה: עם גמר התשלום המלא, כל הזכויות בעיצוב ובקוד יעברו לבעלות הלקוח. "דוב עיצוב ופיתוח" שומרת לעצמה את הזכות להציג את הפרויקט בתיק העבודות שלה (https://dov-portfolio.vercel.app), אלא אם סוכם אחרת בכתב.
